# 🌦️ Weather Analysis Dashboard (Power BI)

## 📌 Overview
This project is a Weather Analysis Dashboard built using Power BI.  
It provides insights into weather conditions such as temperature, humidity, wind speed, and precipitation.

## 🎯 Objective
To analyze weather data and visualize patterns for better understanding and decision-making.

## 📊 Features
- Temperature trends over time
- Humidity analysis
- Wind speed comparison
- Rainfall / precipitation insights
- Interactive filters and slicers

## 🛠️ Tools Used
- Power BI
- Data Visualization
- Data Cleaning

## 📁 Files
- `weather_dashboard.pbix` → Main Power BI file  
- `dataset.csv` → Dataset used  
- `dashboard.png` → Dashboard preview  

## ▶️ How to Use
1. Download the `.pbix` file
2. Open using Power BI Desktop
3. Explore dashboard with filters
